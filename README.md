### To run the project
```
npm install
```
```
npm start
```
** if npm didn't run the dependencies run the code below **

#### Install react router
```
npm i react-router-dom 
```

#### Install axios
```
npm i axios 
```
#### Install json server
```
npm i json-server
```
before use start the server 

#### Run json server
```
npx json-server --watch db.json -- port 5000
```
